# AI Engineer Journey - Session 01 (2026-03-08)

## 📋 학습 요약
오늘 우리는 통계학적 지식을 바탕으로 머신러닝의 기초 워크플로우를 익히고, 실제 모델을 구현하여 성능을 비교했습니다.

### 1. 환경 설정 (Setup)
- **Virtual Environment:** Python venv를 생성하여 독립적인 학습 환경 구축.
- **Libraries:** `scikit-learn`, `pandas`, `numpy`, `matplotlib` 등 필수 라이브러리 설치.
- **GitHub 연동:** GitHub CLI(`gh`)를 설치하고 레포지토리(`ai-engineer-journey`)를 생성하여 코드 버전 관리 시작.

### 2. 구현 모델 및 개념 (Models & Concepts)
- **Linear Regression (선형 회귀):** 
    - 데이터를 관통하는 직선을 찾아 예측하는 가장 기초적인 모델.
    - California Housing 데이터를 사용하여 집값 예측 (R²: ~0.57).
- **Ridge Regression (릿지 회귀):**
    - L2 규제(Regularization)를 추가하여 과적합(Overfitting)을 방지하는 기법 학습.
- **Random Forest (랜덤 포레스트):**
    - 여러 개의 의사결정 나무를 결합한 앙상블 모델.
    - 선형 모델보다 훨씬 높은 성능 확인 (R²: ~0.80).
    - **Feature Importance:** 어떤 변수(`MedInc`, `AveOccup` 등)가 예측에 중요한지 분석하는 법 학습.

### 3. 주요 성과 지표
- **MSE (Mean Squared Error):** 오차의 제곱 평균. 작을수록 정확함.
- **R-squared (R²):** 모델이 데이터의 분산을 얼마나 설명하는지 나타내는 지표 (1에 가까울수록 좋음).
